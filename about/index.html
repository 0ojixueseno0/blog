关于我

关于我

欢迎来到0o酱的个人博客，本博客内容对外开放，转载请标明出处，谢谢谢谢谢谢，别搬孩子的源码去 CSDN 谢谢谢谢。

本站使用 GithubPages + Vercel 服务

我是一名在校学生，一个互联网爱好者，从2010年开始接触电脑。

可以给我打上的标签大致有…福瑞控 沙雕网友 音游人 LGBT（被你发现了…）

我在13年时接触了游戏Minecraft，3年后开始学习Minecraft周边各种元素的知识，服务器，bb建模，材质包，命令方块等已经能够熟练运用。现在的目标是学习Java/Kotlin自行编写一个中小型插件。

目标嘛，虽然我很没主见，不过目前的目标大概是全栈工程师吧，现在是目标为全栈的大菜狗！~

“0o酱”

我为什么叫0o酱？这要从很久以前的一个MC动画制作小组说起了，我的ID是0ojixueseno0，这个ID某种意义上也是致敬了一个喜欢的油管博主，朋友为了叫的顺口，就喊我0o了，为了让名字长那么一点，方便符合“用户名长度最短为3个字符”，于是便叫0o酱了。

后来我的常用ID就改为了Jz0ojiang，这个ID也更符合我现在的名字。

绰号

🤔关于我的绰号还真不少，有叫我圈圈的，有叫蛋蛋的，有叫那个什么酱的（我妈），有叫丘酱的（头像是皮卡丘），还有叫欧欧的（可能以为我ID是大小写的“o”吧）

经历

以下是我会缓慢更新的“这B一生都干了些啥”…

Long long ago…

大概从14年开始，我接触到了Minecraft的命令方块，也许，这就是我学习编程的源头吧。

到了16年，我已经开始研究MC的联机方式，那段时间因为听信了某些教程的鬼话，重置了自家路由器，导致叫了数次宽带维修（根源其实是家用网络不是公网）

17年到18年，我算得上是半个合格的服主了，能够自己配置出一个比较有意思的服务端供朋友们联机了。其实吧，当时完全可以拿这些端圈钱盈利了，主要是因为我比较咕，不愿意去更新，所以最后服务器凉在了第二次内测上…

19年

19年，我开始学习 Skript 插件的语法，这是我第一次系统的学习并编写一个完整功能。同年入学高中，学习了 C 语言基础，同时系统的自学了 html5，CSS3，找到了 Hexo 的建站教程，终于完成了小时候希望有个个人网站的目标，在魔改hexo主题的功能时学习了基础的Js。

下半年，我加入了学校集训校队，开始学习数据恢复技术，在两个多月的培训学习下，参加了省数据恢复与主板维修比赛，最终荣获二等奖，获得参加国赛资格，但因疫情原因导致国赛取消。

20年

20年，我接触了 Ruby 语言，一周内转型到 Python ，开始用 Python 写一些小工具，并以此赚了点外快，给母上大人换了部手机。这是我第一次正经的给我母亲送的较为贵重的礼物。

同期，我配合校队同学，使用Vue.js+Golang编写了体育选课管理系统，虽然最后学校使用了外包平台的选课功能…这也是我最开始接触Vue。

8月，因为在MCBBS水了一个鸡汤楼，就开了一个鸡汤站的坑，起初是使用Ruby作为API，Vue.js作为前端网页。

11月，我用 Python 编写了一款 QQbot ，抛开各种小功能，现在用的最多的是青年大学习的完成统计功能。

12月，使用 Python 重构了鸡汤站的后端API，中途学习了 MySQL 与 Sqlite3 数据库的基础操作与原理，因为部署问题开始研究 Docker。

下半年貌似重写过一遍鸡汤站的前端，用的是 html+js。

21年

21年，因为程序部署，以及 Python 无法编译原因，我开始学习了新的语言—— Golang，使用 Go 语言重构了手上的两个工程 Pichubot 和 Cureword

3月，将鸡汤站后端从 Python 重构到了 Go 语言，因为刚从 Python 跳到 Go ，所以代码比较凌乱，写法比较臃肿。将 Pichubot 从 Python 重构到了 Go 语言。

4月，重构了大部分 Pichubot 的源码，更新版本号为 0.2.0 ，更像一个 SDK 了。

5月，报名参与了第三节江苏省青少年创意编程与智能设计大赛的 Python 创意编程赛项，荣幸拿下了一等奖，国赛因疫情取消。

7月，重写了鸡汤站前端页面，还是用回了 Vue.js ，挖下了一个提交鸡汤的大坑…

8月，重构了鸡汤站后端，写上了提交鸡汤的功能，中途踩了无数个坑… 月底因某项比赛，用 Vue 配合同学写了一个实验室管理系统，我负责前端部分。

9月，开始研究 Electron 与 Vue ，试图挖下更多的坑… 重构了Pichubot的源码，将其写成了一个 go package，可以直接通过Go语言的包管理进行安装，使用 VuePress 为其写了个说明文档。

10月，开始参与联合国儿童基金会月捐计划，入坑光遇，开始为年底的比赛做准备（被迫更新了一堆文章），用 Pichubot 框架做了个机器人.

11月，重构 Pichubot-bot 做了个管理面板，大致样子可以在这里看到其使用文档。

12月，考过了 NCRE 二级python，年底一堆考试的第一场，做了个决定人类命运的小工具-吃点啥，挣了点外快，给自己买了个 freelace pro 耳机，带有主动降噪 真香~

月底，也是 21 年年底，前往南通比赛，拿到了二等奖，比较遗憾的是虽然我的得分已经到了所有人数据恢复得分项中的第三名（一等奖有三名），可是队友因为种种问题没能拿到所负责项目的分数，最终获得了省赛二等奖，与国赛训练就差一名，又一次与国赛擦肩而过~（这家伙就是一个国赛绝缘体）

最后拿到了第二次的国家奖学金，达成成就：连续拿两年国家奖学金~

22年

1月，参加自考考试，三项全部没有挂科，好！（忙比赛那么久竟然没挂科，呜呜呜太好了），趁着打折拿奖学金买了台联想 Yoga14c ，联合国儿童基金会月捐 4 个月目标达成，拿到了纪念戒指和证书，这辈子做的为数不多的善事之一。

2月，B站账号 6 级，过了 18 岁生日了，是个大孩子了，不能装嫩了… 开始学习 Go+Wails2+Vue 的前后端结合用法，尝试写一个生成器工具-MythicGenerator，还在缓慢更新。考过了 WHO 的 Introduction to COVID-19 的某项课程，满分拿到证书。吃了香菜新地 （什么魔鬼味道冰淇淋） 。月底开始学习并制作微信小程序准备参赛~

3月，更新截至此处，待时机成熟，自会更新~嘿嘿嘿
